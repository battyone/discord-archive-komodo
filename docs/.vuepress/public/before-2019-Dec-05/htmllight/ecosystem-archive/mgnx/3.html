@wookie The backend is based on the token DEX CC, where anyone can create, trade and exchange using the colored token system setup by KMD. In regards to the voting system, checks would have to be put in place to prevent clear abuse and multi-voting. Some steps that could be taken: 1) Take steps to monitor and mitigate multi-voting from the actual platform . For example, we could add a Discord alias, a link to a social media account that must tweet out MGN / MGNX at least once. The message signing is flexible and not solely limited to verifying a single piece of information.
