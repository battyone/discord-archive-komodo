[KomodoPlatform/antara-makerbot] Pull request opened: #63 Use crosses instead of sides
The DEX doesn't have a sense of objective quote and base currencies. instead using base to reference the currency being bought. Thus, a buy and sell for a pair doesn't make sense. Because of this, we replace sides by using a Cross and a Pair. A Cross acts like a set of two Assets, without direction; and a Pair acts like a std::pair of two Assets, encoding the information that was previously held by Side.
