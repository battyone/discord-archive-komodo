Hushlist is compatible with any ZEC fork, which means KMD and every KMD asset chain like PIRATE
the most basic form is just using the Zcash protocol memo field, then there are more advanced uses that use "viewing keys" and some features that only some ZEC forks might have/etc. But at it's core, every KMD AC can use Hushlist
