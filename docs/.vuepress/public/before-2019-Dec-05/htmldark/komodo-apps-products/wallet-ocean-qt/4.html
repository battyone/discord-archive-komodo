Hi, All ! Here is some significant changes comes till the end on NN election in Komodo-Qt: 1. We fixed some minor bugs in the wallet in Komodo-Qt, one, according to using of sendmany command from Debug windows -> Console (earlier it caused wallet crash), second with wallets containing z-transactions and few others. 2. Well-known that earlier we had 3 separate branches of 3 different OSes - Linux, Windows and OSX. Now i'm working on get all of this build from one single branch, like original komodod. There was an issues caused by linking OpenSSL 1.1.1a + Qt + other libs, but now all seems solved. This branch is not released yet on Git, but will be published soon and will called static. This also means, that you will able to build static binaries (!) for all 3 OS from single branch. 3. For Windows we will have simple and comfortable NSIS based installer. Linux static binary should work on most distros with LIBC 2.23 (in Ubuntu, it's 16.04 and higher). You can test it and report here will it work on your favorite distro or not. (edited)
