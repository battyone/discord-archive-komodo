CrisF 30-Jun-18 02:08 AM
Hi, in SuperNET/iguana what are the the differences between the scripts in exchanges and dexscripts?
grewalsatinder 30-Jun-18 02:11 AM
exchanges have some example shell scripts and come c files. there is an install script as will, if you execute that it will copy those example shell scripts to dexscripts. reason for that is if you update or change any of the scripts in the exchanges dir and later do git pull you will get issues updating, as git will complain of either discarding local changes or overwriting from git commits updates. for that reason a blank dir dexscripts is there, which is not pushed with update and any changes user makes there doesn't affect git pull updates.
that's the only diff
