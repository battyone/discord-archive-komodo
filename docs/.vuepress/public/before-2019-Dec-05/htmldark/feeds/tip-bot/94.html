Tips the @ mentioned user with the desired amount, e.g. !arrrtip @test123456#7890 1.00

!arrrtip soak <amount> [<@role>]
Soaks the desired amount onto all online and idle users (optionally, within specified role), e.g. !arrrtip soak 5.00, !arrrtip soak 5.00 @Supporters
!arrrtip cannon <amount> [<@role>]
