!arrrtip
Displays this message
!arrrtip info
Displays coin information
!arrrtip balance
Displays your balance
!arrrtip deposit
Displays your deposit address
!arrrtip withdraw <address> <amount>
Withdraws the entered amount to a ARRR address of your choice, e.g. !arrrtip withdraw GLM3n7yMFmYkas2AgHKqE6EMJW5VsUXtQn 5.20 Note: Minimal amount to withdraw: 1 ARRR. A withdrawal fee of 0.2 ARRR will be automatically deducted from the amount.
!arrrtip <@user> <amount> [private]
Tips the @ mentioned user with the desired amount, e.g. !arrrtip @test123456#7890 1.00
!arrrtip squall <amount> [<@role>]
Rains the desired amount onto 100 random online users (optionally, within specified role), e.g. !arrrtip squall 50.00, !arrrtip squall 50.00 @Supporters
!arrrtip soak <amount> [<@role>]
Soaks the desired amount onto 100 random online and idle users (optionally, within specified role), e.g. !arrrtip soak 50.00, !arrrtip soak 50.00 @Supporters
!arrrtip cannon <amount> [<@role>]
Tips 2-5 random lucky online users with the amount (optionally, within specified role), e.g. !arrrtip cannon 5.00, !arrrtip cannon 5.00 @Supporters
!arrrtip faucet
Gets an amount from the faucet (applicable every 4 hours)
!arrrtip lottery
Display information about current lottery cycle
!arrrtip lotterymanual
Displays a short lottery manual
!arrrtip lotterytickets [<@cycleNumber>]
Display information about all tickets bought in specified lottery cycle (current if left empty)
!arrrtip buyticket <amount> [private]
Buys lottery tickets for the specified amount if a lottery cycle is active, e.g. !arrrtip buyticket 5.00
!arrrtip donate <amount> [private]
Donates to the current lottery cycle without buying a ticket, e.g. !arrrtip donate 5.00
