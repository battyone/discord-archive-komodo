The orange block at fork index 0, O0, is endorsed by two PoP publications which occur at index 1 of the SP blockchain, and one PoP publication at index 2 in the SP blockchain, so its score is 2(1/((1-1+1)(1-1+1))) + 1(1/((2-1+1)(2-1+1))) = 225; O0 = 225.
