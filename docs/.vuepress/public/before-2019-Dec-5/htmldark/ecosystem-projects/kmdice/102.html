TonyL 29-Dec-18 04:02 AM
@HenryChinaski You have to run a komodo daemon locally for asset chain on which you want to play For example you can start KMDICE native mode in Agama Then your config with RPC username, password and port will be in %AppData%/Komodo/KMDICE/KMDICE.conf (edited)
If you run daemon locally as IP use 127.0.0.1
Connection details will be saved to file after successful connection so you'll not need to input it again. Actually daemon can be started anywhere (that's why I left IP field). For example I have one node on Ubuntu VPS and can play on any of my PCs from same "account" (actually wallet) without daemon installation/syncing. But if you run it on remote server - additional RPC whitelisting by IP needed
