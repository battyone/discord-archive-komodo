@everyone Friendly reminder that we will upgrade the Komodo code base to version 0.5.0, dubbed Obsidian Dragon on December 20, 2019, at block height 1670000. As a precaution, we have also advised all services and Exchanges to pause the deposits and withdrawals from Dec. 19th - 21st, 2019. This is a mandatory platform upgrade, so all miners, exchanges, and individuals running full-seed nodes must update to the latest software before block 1670000. Those who do not update before December 20 will not run on the correct network.
