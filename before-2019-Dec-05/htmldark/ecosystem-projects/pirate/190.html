!tip distribute <preview|perform>
This command takes an uploaded .csv file with two columns: ARRR wallet address, amount and performs the airdrop to the listed addresses. Individual amounts per address are allowed. .csv file delimiter must be , (comma sign) - check your regional settings for list delimiter if it is not. The easiest way to issue this command is: drag and drop the .csv file onto the discord chat window, and type in the command in the additional info window that opens. distribute preview will parse the file and perform all validations. distribute perform will parse the file, perform all validations, and execute the payout. Note: always have a little extra in your tipbot wallet because transaction fee is not subtracted from the individual amounts and is difficult to estimate due to variable number of inputs.

!tip autorain <amount> [<@role>]
Rains the desired amount from the autoraining pot onto all online users (optionally, within specified role), e.g. !tip autorain 10.00, !tip autorain 10.00 @Supporters. This command can be automated.

Soaks the desired amount from the autoraining pot onto all online and idle users (optionally, within specified role), e.g. !tip soak 5.00, !tip soak 5.00 @Supporters. This command can be automated.
