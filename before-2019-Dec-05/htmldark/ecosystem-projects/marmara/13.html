@Paro I updated code in my repo (https://github.com/dimxy/komodo/tree/marmara-v1-1-opret-v2), fixed some issues with staking for boosted coins. I recommend starting a new chain to check boosted coins PoS generation. for a new chain, to see how boosted coins generate blocks, make sure that all the segids have equal utxo values, for example make them with command 'marmaralock64 640 1', this will create 64 segids with one utxo in each of 10 coins.
