[artemii235/SuperNET] Issue opened: #289 Remove "A0 B0" keypairs from swap protocol.
The temporary "A0 B0" key pairs are used for UTXO coins input signing, e.g maker payment:
OP_IF <now + LOCKTIME*2> OP_CLTV OP_DROP <bob_pubB0> OP_CHECKSIG OP_ELSE OP_SIZE 32 OP_EQUALVERIFY OP_HASH160 <hash(bob_privN)> OP_EQUALVERIFY <alice_pubA0> OP_CHECKSIG OP_ENDIF
It looks like it's more an artifact of Tier Nolan protocol that is not used anymore, we can safely remove it.
